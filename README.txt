== Description ==

WP Currency Rate is a plugin for currency converter. It's using yahoo currency API to get latest currency rate. WP Currency Rate supports 157 currency. 



== Installation ==

1. Upload `wp-currency-rate` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set your base currency and display currency from 'WP Currency Rate' from menu
4. Add Currency Rate widget

== Frequently Asked Questions ==

= How many currency it will support =

It will support 157 currency
